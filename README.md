# Flow‑Guided Generative Drawing Engine

This repository contains the design and eventually the source code for a generative drawing tool built with **Processing (Java)**. Users manipulate a vector field with brushes; autonomous "bots" then traverse the field and leave trails that form the final artwork.

## Purpose
- Quick prototype implementation using Java/Processing
- Follow the provided roadmap and software requirements without changing core subsystems
- Expose configuration for normalization and radar sampling

## Getting Started
1. Install Processing (latest 4.x) or set up a Java project that depends on `core.jar` from Processing.
2. Open `MainApp.pde` in the Processing IDE or import into your preferred Java IDE.
3. Verify that a blank P2D window opens and that camera controls (zoom/pan) work.
4. Follow the phased development plan from `deveopment_roadmap.txt` to add features incrementally.

- `Software requirements` – detailed requirements and UI description (original Spanish)

## Notes

- Configuration values (thresholds, sample counts) should be trivially adjustable once the core systems exist.
